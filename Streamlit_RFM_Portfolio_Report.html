1. Strategy & Product Presentation

The Gap: Advanced data models in standalone Jupyter notebooks often go unused because marketing stakeholders and business leads lack the technical background to run code directly.

The Operational Bridge: Built an interactive web workspace that lets product or growth teams upload transaction ledgers and instantly generate customer cohort distributions without writing a single line of code.

2. Architecture & Interactive Core Elements

The application layout leverages interactive UI elements to help users drill down into specific data points:

  • File Ingestion Interface: Includes an integrated drag-and-drop tool for uploading CSV files, automatically validating required columns (CustomerID, InvoiceDate, UnitPrice).
  • Interactive Metric Sliders: Added custom filter handles that let users adjust scoring ranges to fine-tune cohort classification definitions on the fly.
  • Dynamic Chart Rendering: Uses interactive charting libraries to automatically update distribution plots whenever the underlying source criteria change.
